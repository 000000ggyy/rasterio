Rasterio Documentation¶

Rasterio is for Python programmers and command line users who want to read, write and manipulate geospatial raster datasets.

Rasterio employs GDAL under the hood for file I/O and raster formatting. Its functions typically accept and return Numpy ndarrays. Rasterio is designed to make working with geospatial raster data more productive and more fun.

Install with pip (see the complete installation docs )

pip install rasterio

And an example of use in python

import rasterio

with rasterio.open('data.tif') as src:
    array = src.read()
